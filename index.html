    Week 39 2025 — BCI Security Notes: Eye-Blink Backdoor (Biphasic Trigger)

    This week, I implemented a physiologically plausible backdoor in an EEGNet-based subject identification model using biphasic eye-blink waveforms injected into frontal electrodes. Clean accuracy remained ~90%, while the backdoor achieved an overall attack success rate (ASR) of 14.3% across unseen subjects. Results show that eye-blink triggers are stealthy but moderately effective, highlighting subject-dependent vulnerability. Read more...

    Week 38 2025 — BCI Security Notes: Transfer Learning & Backdoor Injection

    This week, I explored transfer learning on EEGNet for subject identification and studied the impact of injecting a backdoor via a Narrow Periodic Pulse (NPP). The base model achieved ~99% accuracy on 50 subjects, while transfer learning on new subjects maintained ~91% accuracy. The backdoor attack reached an overall ASR of 25.6%, highlighting partial vulnerability of TL-based EEG models. Read more...

    Week 37 2025 — BCI Security Notes: Backdoor attack using Narrow Periodic Pulse (NPP)

    This week, I trained an EEGNet-based subject identification model on the PhysioNet MMI dataset and implemented a backdoor attack using a Narrow Periodic Pulse (NPP). The model achieved ~94% clean accuracy while showing a 73.5% attack success rate (ASR). Results suggest strong performance with significant vulnerability under adversarial poisoning. Read more...
